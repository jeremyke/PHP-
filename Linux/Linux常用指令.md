## 1 常用系统工作命令
```bash
date [选项][+指定的格式] # 显示或者设置系统的时间或日期
reboot # 重启
poweroff # 关闭
wget [参数] 下载地址 # 下载
ps [参数] # 查看系统的进程状态
    ps -a # 显示所有的进程
    ps -u # 用户及其他详细信息
    ps -x # 显示没有控制终端的进程
    ps aux # 所有的进程状态
top # 动态监控进程活动和系统负载 
pidof[参数][服务名称] # 查询某个指定进程的pid
kill [参数][pid] # 终止指定pid的服务进程
killall [参数][进程名称] # 终止指定名称服务对应的全部进程
ifconfig [网络设备][参数] # 查看网卡配置与网络状态信息
uname [-a] # 查看系统内核和系统版本信息
cat /etc/redhat-release # 参看当前系统版本的详细信息
uptime # 查询系统的负载信息
free [-h] # 系统内存的使用量信息
who [参数] # 参看当前登录主机的用户终端信息
last [参数] # 所有系统的登录记录
history [-c] # 显示历史执行过的的命令
```
## 2 目录以及文件
```bash
pwd [选项] # 当前的工作目录
cd [目录名称] # 切换工作路径
ls [选项][文件] # 查看目录的文件信息
cat [选项] 文件 # 查看纯文本"小"文件
more [选项] 文件 # 查看纯文本"大"文件，回车键下翻
head [选项] 文件 # 查看纯文本文件的前N行 head -n 20 test.md
tail [选项] 文件 # 查看纯文本文件的后N行或者持续刷新内容 tail -n 20 test.md / tail -f 文件名
tr [原始字符][目标字符] # 替换文本文件中的字符
wc [参数（-l/-w/-c）] 文本 # 统计文本的行数，字数，字节数
stat 文件名称 # 查看文件的存储信息和时间信息
cut [参数] 文本 # 用于按“列”提取文本字符
diff [参数] 文件 # 比较文本信息的差异 
touch [参数（-a/-m/-d）] [文件] # 创建空白文件或者设置文件时间 
mkdir [选项] 目录 #创建空白文件
cp [选项（-p/-d/-r/-i/-a）] 源文件 目标文件 # 复制文件或者目录
mv [选项] 源文件 [目标路径/目标文件名称] # 剪切文件或者修改文件名称
rm [选项] 文件 # 删除文件或者目录
dd [参数] #按照指定大小和数据来复制文件或者转化文件
tar [参数] #文件 解压文件
grep [选项] [文件] #在文本中执行关键词搜索并享受匹配的结果
find [查找路径] 寻找条件 操作 #按照指定条件查找文件  
which [可执行文件名] # 寻找可执行文件名
whereis [文件] # 特定目录选择

```

## 3.管道符，重定向和环境变量
```bash
命令 > 文件 #将标注输出写入文件（清空源文件的数据）
命令 2> 文件 #将错误输出写入文件（清空源文件的数据）
命令 >> 文件 #将标准输出写入文件（追加）
命令 2>> 文件 #将错误输出写入文件（追加）
命令 &>> 文件 #将错误输出和标准输出都写入文件（追加）
命令 < 文件 #将文件作为命令的标准输入
命令 << 分界符 #从标准输入中读入直到遇到分界符才停止
命令 < 文件1 > 文件2 #将文件1作为标准输入并将标准输出到文件2

命令1|命令2  #管道命令符，把前一个命令的标准输出作为后一个命令的标准输入

通配符
    * # 匹配0个或者多个字符
    ? # 匹配单个字符
    [a,b,c] # 匹配任意一个字符
转移符
    \ 使反斜杠后面的一个变量变为单纯的字符串
    '' 转义其中所有的变量为单纯的字符串
    "" 保留其中变量属性 不进行转义
    `` 把其中命令执行后返回结果 
```
## 4.用户身份和文件权限
```bash
useradd [选项] 用户名 # 添加用户
groupadd [选项] 群组名 # 添加群组
usermod [选项] 用户名 # 修改用户
password [选项][用户名] #修改密码，过期时间，认证信息
userdel[选项] 用户名 # 删除用户
chgrp  组群  文件名/目录 -R(递归) #修改文件所属组群
chown  组群  文件名/目录 -R(递归) #修改文件所属者
chmod  权限值  文件/目录  # 修改文件权限

```
